# Public Folder

This folder contains static assets for the portfolio.

## Photo Placement

**Place your photo here as `photo.jpg`**

Your photo should be:
- Named exactly: `photo.jpg` (or update the path in `src/components/Hero.jsx`)
- Recommended size: 300x300 pixels or larger (square aspect ratio preferred)
- Format: JPG, PNG, or WebP

The portfolio will automatically use this photo in the Hero section. If the photo is not found, it will show a placeholder with initials "SM".
