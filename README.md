# Shawana Maliha - Personal Portfolio

A modern, professional portfolio website showcasing Full-Stack Development, Software Quality Assurance, and Machine Learning expertise.

## Features

- 🎨 Modern, minimalist UI with light blue theme
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive design
- ✨ Smooth scroll animations
- 🔍 Project filtering by category
- ⚡ Built with React and Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

1. Add your photo as `public/photo.jpg`
2. Update contact information in `src/components/Contact.jsx`
3. Update social media links in `src/components/Contact.jsx`
4. Customize project details in `src/components/Projects.jsx`
5. Update experience and leadership details in `src/components/Experience.jsx`
